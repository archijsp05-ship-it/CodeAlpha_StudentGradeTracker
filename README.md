# 🎓 Student Grade Tracker

A Java application that helps manage student grades efficiently. This project is available in both **Console-Based** and **GUI-Based (Java Swing)** versions. It allows users to add student records, calculate average, highest, and lowest marks, and generate a summary report.

---

## 📌 Features

- ✅ Add student names and marks
- ✅ Store data using ArrayList
- ✅ Calculate average marks
- ✅ Find highest score
- ✅ Find lowest score
- ✅ Display all student records
- ✅ Console-based application
- ✅ GUI version using Java Swing
- ✅ Beginner-friendly OOP implementation

## 🚀 How to Run

### Console Version

1. Open the project in IntelliJ IDEA.
2. Run `StudentGradeTracker.java`.
3. Enter student names and marks.
4. View the generated report.

### GUI Version

1. Open the project in IntelliJ IDEA.
2. Run `StudentGradeTrackerGUI.java`.
3. Enter student details.
4. Click **Add Student**.
5. Click **Show Report** to view results.
